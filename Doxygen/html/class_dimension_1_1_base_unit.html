Detailed Description

template<typename Unit>
class Dimension::BaseUnit< Unit >

A base class representing a unit.

This abstract class represents a Unit, such as Meters, Seconds, Grams, etc.

Template Parameters
UnitThe derived unit, using CRTP

Convert the input value from this unit to the given unit.

This will use the direct conversion if one is provided, otherwise it will convert to the primary unit, then to the given unit. Conversions are inverted if isNumerator is false.

Validate all conversions.

This function should be called as part of the initialization step of each derived Unit type. It should be given the vector of all Units for the Unit type at hand, as well as the Primary Unit of that Dimension

Parameters
[in]UnitVectorThe vector of all Units within this dimension
[in]PrimaryUnitThe "primary" unit for this dimension. Every unit must have a conversion to and from this unit. While other conversions are allowed, these conversions are mandatory.
